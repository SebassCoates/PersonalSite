A New Personal Website

Last summer, I created my first personal website (the source code is still available, but it is not hosted anymore). The objective was to showcase my projects, research, and resume. I created all the style from scratch, as it was my first real web project and I didn't want to rely on a framework. The end result was a navigable website that was kind of responsive. At the same time, it was difficult to maintain and the overall design was lacking. While it was a useful project as an introduction to web development, it soon became time for a new site.

Today, I've completed a new personal website! Instead of building it completely from scratch this time, I started with a handy Bootstrap template. The goals of this website are similar to my initial goals: to share my projects and research. I now have the additional goal of updating the site somewhat regularly with posts (like this one)! My hope is that this will allow me to better document progress on my work. This will also be a convenient platform to document my upcoming study abroad experience at Oxford.

Ultimately, I'm quite happy with the final result! This website is a vast improvement over my previous site. I'm excited to keep the website content up-to-date as I work on new projects and research. Hopefully I'll have time to post updates (somewhat) regularly!
